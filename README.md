# Portfolio:

## 🚀 Features

-   **Semantic Architecture:** Built using meaningful HTML tags (`<header>`, `<article>`, `<footer>`) to ensure high SEO rankings and structural clarity.
-   **Full Interactivity:**
    -   **Project Filtering:** Dynamic categorization of portfolio work.
    -   **Lightbox Gallery:** Modal view for detailed project inspection.
    -   **Mobile Navigation:** Fully functional hamburger menu for smaller viewports.
    -   **Smooth Scrolling:** Enhanced UX for single-page navigation.
-   **Accessibility (A11y):** ARIA roles, high-contrast color schemes, and keyboard-friendly interactive elements.
-   **Responsive Design:** Fluid layouts optimized for Desktop, Tablet, and Mobile devices (768px breakpoint).

## 🛠️ Built With

* **HTML5** - Structure and Semantic markup.
* **CSS3** - Custom styling, Flexbox grid, and Media Queries.
* **JavaScript (Vanilla)** - DOM manipulation and Form validation.
* **Microsoft Copilot** - AI-assisted debugging and code optimization.
